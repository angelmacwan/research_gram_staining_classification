This script trains and evaluates the final selected model for Gram staining image classification.
It uses robust data augmentation and logs comprehensive metrics, including per-class and overall performance.

How to Use:

1. Prepare your TRAIN and TEST folders with subfolders for each class (GNB, GNC, GPB, GPC).
2. Adjust script parameters if needed:
   - model_name, epochs, augmentation settings, etc.
3. Run the script:
   - The script will train and evaluate the final model, saving results and metrics to the LOGS folder.
